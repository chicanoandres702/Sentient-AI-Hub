# Sentient Agent Hub UI Extension

This folder contains the React UI for orchestrating and monitoring all agents and CI/CD gates.

## Features
- Trigger main-agent.js workflows
- Display gate status and audit logs
- View results from all subagents

## How to Use
1. Scaffold a React app here (e.g., with `npx create-react-app .`).
2. Add components for gate status, workflow triggers, and log display.
3. Connect to backend agents via API or direct Node.js calls.
4. Run with `npm start` and open http://localhost:3000.
