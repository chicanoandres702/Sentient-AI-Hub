# Frontend for Sentient Agent Hub GitHub App Dashboard

## Features
- Native dashboard UI for your repo
- GitHub colors and excessive neon blue glow
- Shows gate status, workflow trigger, audit logs

## Setup
1. Run `npm install`
2. Start with `npm start`
3. Connects to backend at `/api/`

## Customization
- Edit `src/githubTheme.js` for colors and glow
- UI components: GateStatus, WorkflowTrigger, AuditLog
